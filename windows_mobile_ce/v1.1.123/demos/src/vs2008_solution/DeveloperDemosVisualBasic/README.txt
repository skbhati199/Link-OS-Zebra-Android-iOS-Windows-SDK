If deploying to a Motorola(R)/Symbol terminal, you must copy the 
<install_path>\windows_mobile_ce\<version>\lib\ZebraPlatformUtil.dll
and any additional Motorola libraries described in the <install_path>\readme.html
to the %CSIDL_PROGRAM_FILES%\VbDevDemos directory.
	
If deploying to a device that requires USB host support, you must
copy the <install_path>\windows_mobile_ce\<version>\lib\ZebraUsb.dll
to the %CSIDL_PROGRAM_FILES%\VbDevDemos directory.
